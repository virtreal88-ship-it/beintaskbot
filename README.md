# BeinTaskBot

Telegram-бот для Bein Systems с интеграцией Kommo CRM, AI-обработкой сообщений и полной автоматизацией.

## Возможности

- AI-анализ текстовых и голосовых сообщений (OpenAI)
- Интеграция с Kommo CRM (задачи, контакты, сделки, этапы воронки)
- Автоматические переходы по этапам воронки
- Транскрипция голосовых сообщений
- Фоновые уведомления (дедлайны задач, утренний дайджест, зависшие сделки)
- Ролевая система (Admin, Şamil, Soltan)
- Интерфейс на азербайджанском языке

## Деплой на Railway

### Переменные окружения

| Переменная | Описание |
|---|---|
| `TELEGRAM_TOKEN` | Токен Telegram-бота |
| `KOMMO_TOKEN` | Bearer-токен Kommo CRM |
| `OPENAI_API_KEY` | Ключ OpenAI API |
| `OPENAI_API_BASE` | Base URL для OpenAI API |

### Запуск

Бот запускается как **worker** (polling), не как web-сервис.

```
worker: python bot.py
```

## Системные зависимости

Для транскрипции голосовых сообщений требуется `ffmpeg` (устанавливается автоматически через Nixpacks на Railway).
